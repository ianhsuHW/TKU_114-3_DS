# ATM 提款課堂實作

## 問題拆解：input、process、output

### 1. input
需要輸入什麼：
- 帳戶餘額
- 提款金額

### 2. process
要判斷什麼：
- 提款金額是否大於 0
- 提款金額是否小於等於帳戶餘額

要更新哪些變數：
- 如果提款成功，更新帳戶餘額：
  - 新餘額 = 舊餘額 - 提款金額

### 3. output
要輸出什麼結果：
- 如果提款成功：顯示「提款成功」
- 如果提款失敗：顯示「提款失敗，餘額不足或金額不正確」
- 最後顯示目前餘額

## 簡短答案整理
- input：帳戶餘額、提款金額
- process：檢查提款金額是否合法，並更新餘額
- output：顯示提款結果與目前餘額
